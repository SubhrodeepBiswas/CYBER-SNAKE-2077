# 🐍 CYBER SNAKE 2077

A futuristic **Snake Game** built with **Python Turtle Graphics** and **Tkinter**, featuring a neon cyberpunk-inspired interface, animated gameplay, start and restart screens, score tracking, and smooth movement.

![Python](https://img.shields.io/badge/Python-3.x-blue?logo=python)
![Turtle](https://img.shields.io/badge/Turtle-Graphics-green)
![Tkinter](https://img.shields.io/badge/Tkinter-GUI-orange)

---

## ✨ Features

* 🎮 Smooth Snake movement
* 🌌 Cyberpunk-themed UI
* ▶️ Start screen with Play button
* 🔄 Restart button after game over
* 🍎 Random food generation
* 📈 Live score display
* 🏆 High score tracking
* 💀 Wall collision detection
* 🐍 Self-collision detection
* ⚡ Non-blocking game loop using Tkinter's `after()`
* 🎨 Neon colors and futuristic design

```bash

```

### 2. Navigate into the project folder

```bash
cd Cyber-Snake-2077
```

### 3. Run the game

```bash
python main.py
```

## Game Rules

* Eat the pink food to grow longer.
* Every food increases your score by **1**.
* Avoid touching the walls.
* Avoid colliding with your own tail.
* Try to beat your highest score!
